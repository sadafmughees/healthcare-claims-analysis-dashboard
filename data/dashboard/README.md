# UK Claims Analytics Dashboard

## Project Overview
This project presents an interactive analytics dashboard for synthetic UK insurance claims data.

## Features
- Claim approval analysis
- Average claim amount by country
- Fraud trend visualization
- Interactive Plotly charts

## Technologies Used
- Python
- Dash
- Plotly
- Pandas

## Run Dashboard

python app.py
